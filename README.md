An interactive Power BI dashboard designed to analyze hospital operations, patient trends, doctor performance, and financial metrics using real-world healthcare data.
This project focuses on data-driven decision-making through clean visuals and meaningful KPIs.

🧩 Problem Statement

Hospitals generate large volumes of data across departments such as patients, doctors, finance, and staff.
The challenge is to convert raw data into actionable insights that help management:

Improve patient care

Optimize doctor workload

Track revenue and expenses

Monitor staff efficiency

🎯 Objectives

Analyze hospital performance using KPIs

Provide department-wise insights

Enable quick decision-making through interactive dashboards

Apply data cleaning, transformation, and modeling techniques

🛠 Tools & Technologies

Excel

SQL

DAX

Power Query

Data Cleaning & Transformation
